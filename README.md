# Instamart-Sales-Performance-Dashboard
Instamart Sales Performance Dashboard built using Power BI and SQL, providing actionable insights into sales trends, top-performing products, customer behavior, and delivery performance. The dashboard enables data-driven decision-making through interactive visualizations and optimized data modeling.
